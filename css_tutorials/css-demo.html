CSS - Cascading Stylesheet

CSS is a way to design the webpage. We work on the basic fundamental properties of look and feel for the web page. Following are some properties we consider while styling the webpage -

  • Colors
  • Dimensions
  • Space
  • Shape
  • Fonts
  • Positioning
  • Responsiveness
  • Transformations
  • Transitions and Animations

Syntax to apply the CSS - 

            property1 : value;
            property2 : value;
            ...
            propertyN : value;

There are 3 types of CSS

  1. Inline CSS - The CSS which is applicable on any specific tag. Its written in the body of tag using style attribute.
  2. Internal CSS - The CSS which is applicable on all selected elements present on the web page. Its written in the head section using style tag.
  3. External CSS - The CSS which is applicable on multiple webpages for all selected elements. This is written in a separate file saved with .css extension. And it is linked to the respective web pages using tag inside the head tag of webpage.
