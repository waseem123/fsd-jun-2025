Pseudo Classes

Pseudo classes are used for changing the effect of any HTML element.

There are 4 Pseudo Classes in CSS

  1. Link
  2. Active
  3. Visited
  4. Hover
Syntax to apply the Pseudo Class
        selector : pseudo_class_name{
            property 1 : value;
            property 2 : value;
            ...
            property N : value;
        }
